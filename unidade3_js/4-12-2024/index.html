Calculadora em JavaScript!

Atividade Opcional - 1.5pt adicional na N2 ou N3

Faça um formulário para cadastrar um cliente.

Este formulário deve contar campos para cadastrar o nome, email, telefone e observação

Após o cadastro, exiba os dados do cliente em uma tabela e sumir o formulário.

A tabela só deve aparecer após clicar no botão enviar os dados

Abaixo da tabela deve contar um botão voltar, para aparecer o formulario e sumir a tabela

Faça tudo com animação no css
